Interactive graph query formulation

Go to Editor

We

present Orion, a visual interface for interactive graph query formulation. Large heterogeneous graphs like Freebase and DBpedia are a great source of information. But their complex schema makes it difficult for users to specify exact queries. Orion is a graphical interactive interface that helps schema agnostic users formulate exact query graphs. Orion automatically recommends relevant edges to add to the query graph that can be either accepted or ignored by users. Users can also manually add new nodes and edges to the query graph using simple mouse actions!


Please use Orion to make your query formulation process easier!
